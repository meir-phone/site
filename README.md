# מערכת ניהול מרחב קולי

מערכת ניהול תלמידים והאזנות למרחב הקולי עם עיצוב מודרני בצבעי ירוק בהיר ומרם בד פשתן.

## קבצים במערכת

### דפי HTML
- **index.html** - דף ראשי עם רשימת כל התלמידים
- **student-detail.html** - דף פרטי תלמידה עם היסטוריית האזנות
- **login_details.html** - דף כניסות קיים (נשמר)

### קבצי עיצוב וקוד
- **styles.css** - עיצוב מלא עם Fluent UI Icons
- **students.js** - לוגיקה לדף התלמידים
- **student-detail.js** - לוגיקה לדף פרטי תלמידה

### קבצי נתונים
- **תלמידים.xlsx** - קובץ Excel עם פרטי התלמידים
- **ListeningOk.ini** - קובץ עם נתוני האזנות

## תכונות

### דף ראשי (index.html)
- ✅ רשימת כל התלמידים בכרטיסים מעוצבים
- ✅ חיפוש לפי שם, שכבה, כיתה או ת.ז
- ✅ סינון לפי שכבות (ט', י', יא', יב')
- ✅ סטטיסטיקות: סך תלמידים וסך האזנות
- ✅ כל כרטיס מציג: שם, שכבה, כיתה, טלפון, מספר האזנות
- ✅ לחיצה על כרטיס מעבירה לדף פרטי התלמידה

### דף פרטי תלמידה (student-detail.html)
- ✅ פרופיל מלא של התלמידה
- ✅ היסטוריית האזנות מלאה
- ✅ כל האזנה מציגה: קובץ, נתיב, תאריך ושעה
- ✅ מיון לפי תאריך (החדשים ביותר למעלה)
- ✅ כפתור חזרה לדף הראשי

### עיצוב
- 🎨 פלטת צבעים: ירוק בהיר (#90C695) ומרם בד פשתן (#F4EFE6)
- 🎨 Fluent UI Icons מ-Microsoft
- 🎨 עיצוב רספונסיבי למובייל
- 🎨 אנימציות חלקות
- 🎨 הדר קבוע עם לוגו וניווט

## הפעלה

1. **פתח את הקבצים בדפדפן**
   - פתח את `index.html` בדפדפן
   - או הרץ שרת מקומי:
   ```bash
   python -m http.server 8000
   ```
   ואז פתח: http://localhost:8000

2. **וודא שקבצי הנתונים במקום**
   - `תלמידים.xlsx` - חייב להכיל עמודות: ת.ז, שם, שכבה, כיתה, טלפון
   - `ListeningOk.ini` - בפורמט: teudat_zehut_ID-/path/-file-Date-YYYY-MM-DD-Hour-HH-MM-SS

## מבנה קובץ Excel

הקובץ `תלמידים.xlsx` צריך להכיל את העמודות הבאות:
- **ת.ז** או **תעודת זהות** - מספר תעודת זהות
- **שם** או **שם מלא** - שם התלמידה
- **שכבה** - שכבת הלימוד (ט, י, יא, יב)
- **כיתה** - מספר הכיתה
- **טלפון** - מספר טלפון (אופציונלי)

## מבנה קובץ INI

הקובץ `ListeningOk.ini` בפורמט:
```
teudat_zehut_219578937-/1/3/3/-001=yes-Date-2026-03-09-Hour-18-54-25
```

כל שורה מכילה:
- מזהה תלמידה (ת.ז)
- נתיב הקובץ
- שם הקובץ
- תאריך
- שעה
